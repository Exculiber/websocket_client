WebSocket 探测工具 - Macos 版本

🚀 快速开始:
  ./websocket-probe-macos-arm64 wss://echo.websocket.org

📖 查看帮助:
  ./websocket-probe-macos-arm64 --help

🎮 交互式模式:
  ./websocket-probe-macos-arm64 wss://your-server/ws --mode interactive

🔧 跳过SSL验证:
  ./websocket-probe-macos-arm64 wss://192.168.1.100/ws --skip-ssl-verify

🔍 调试模式:
  ./websocket-probe-macos-arm64 wss://your-server/ws --debug
